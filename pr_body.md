🛠️ 리뷰 및 로컬 테스트 방법
1. 브랜치 체크아웃 후 프론트엔드(`cd frontend && npm run dev`) 및 백엔드 기동
2. 분석 페이지에서 특정 파일을 컨텍스트로 지정 후 채팅 질문 시도
3. 채팅 답변 생성 중에 "멀티에이전트 자율 탐색" 토글이 자동으로 열리는지 확인
4. 답변 생성이 끝나면 토글이 자동으로 닫히고, 메시지에 `CONTEXT: 파일명` 칩이 표시되는지 확인
